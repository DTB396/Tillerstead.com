ðŸ“± Responsive Behavior

Mobile Optimization

Automatic Adjustments:

900px and below:

  • Reduce tile pattern sizes (100px â†’ 80px)
  • Reduce checkerboard size (40px â†’ 30px)
  • Reduce geometric sizes (60px â†’ 40px)
  • Reduce shadow offsets (5px â†’ 3px)

600px and below:

  • Further reduce tile sizes (80px â†’ 60px)
  • Lower pattern opacity (8% â†’ 5%)
  • Simplify card borders (3px â†’ 2px)
  • Hide Memphis geometric accents (performance)
  • Reduce border chunky sizes

Performance:

  • Patterns disabled on mobile for faster load
  • Simplified shadows and borders
  • Reduced opacity overlays

â™¿ Accessibility

WCAG AA Compliance

Automatic Support:

Reduced Motion:

@media (prefers-reduced-motion: reduce) {
  /* All animations disabled */
  /* Button/card hovers don't transform */
}

High Contrast:

@media (prefers-contrast: high) {
  /* Patterns disabled */
  /* Borders increased to 4px */
  /* Overlays removed */
}

Text Contrast:

  • All retro colors maintain 4.5:1 contrast minimum
  • Pattern overlays kept at 5-15% to avoid interfering with text
  • Buttons have clear focus states

Creating Custom Retro Colors
