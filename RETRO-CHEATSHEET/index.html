‚úÖ Usage Rules

DO:

  • ‚úÖ Use as accents (8-15% opacity)
  • ‚úÖ Apply to CTAs, cards, dividers
  • ‚úÖ Combine with emerald/gold brand colors
  • ‚úÖ Maintain WCAG AA contrast (4.5:1)
  • ‚úÖ Test on mobile devices

DON‚ÄôT:

  • ‚ùå Use as primary text color
  • ‚ùå Exceed 20% opacity on patterns
  • ‚ùå Use 3+ retro colors per section
  • ‚ùå Compromise accessibility
  • ‚ùå Apply to every element

üì± Responsive Behavior

900px and below:

  • Tile patterns reduce to 80px
  • Shadows reduce to 3px
  • Checkerboard reduces to 30px

600px and below:

  • Tile patterns reduce to 60px
  • Pattern opacity reduces to 5%
  • Borders reduce to 2px
  • Memphis accents hidden (performance)

‚ôø Accessibility

Reduced Motion:

@media (prefers-reduced-motion: reduce) {
  /* All animations disabled */
}

High Contrast:

@media (prefers-contrast: high) {
  /* Patterns disabled, borders increased */
}
