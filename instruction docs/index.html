Curbless Shower Documentation Set

This folder provides the stewardship-focused guidance Tillerstead uses to plan and deliver true curbless (zero-threshold) showers with durable workmanship, transparent client communication, and verifiable records. Each document supports contractors, homeowners, and property managers in aligning field execution with long-term community trust.

Contents

  • Curbless Shower Rebuild Reference – Full build sequence, definitions, planning equations, and failure cautions.
  • Drain and Waterproofing Options – System selection guidance for drains, membranes, and compatibility checks.
  • Inspection and Risk Notes – Compliance awareness, slope verification references, and documentation practices.
  • Templates
    • Jobsite Checklist
    • Flood Test Log
    • Client Explanation (Plain English)

Planning-tool notice

All equations, tables, and templates herein are planning aids intended for calculators, estimators, and education widgets. Every measurement, structural modification, and inspection item must be verified in the field against manufacturer instructions and local jurisdiction requirements before work proceeds.

Tillerstead’s stewardship approach prioritizes craftsmanship, measured communication, and community confidence—document each step, verify locally, and build for lasting performance.
